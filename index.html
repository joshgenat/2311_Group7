About MusicXML

MusicXML can be used to precisely denote a piece of music. This format is supported by many music software and can display the song in an easy to read fashion, edit on the go, play the song, and many more features! Tab2XML is an application that allows users to take a tablature and convert it into a more flexible file, MusicXMLâ€”allowing users to adjust the piece of music to their liking and allow for better transferability.

Why MusicXML?

Our objective is to develop a software system that allows the user to input a tablature of their choice and receive the MusicXML file for that particular text tab. This project will convert a text tab containing guitar or drum tablature into a MusicXML file. The system is designed to ease file conversion and allow a convenient application for music enthusiasts looking to utilize the efficiency of MusicXML.

Getting Started

To begin using the Converter, please download the Java zip file and open the file in Eclipse. The software will be run through Gradle.
